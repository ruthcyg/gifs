Understanding Django's `raw()` Method

Method Signature: Manager.raw(raw_query, params=(), translations=None)

raw_query: SQL Query String

Description: This parameter is the actual SQL query string that Django will execute. You can use any valid SQL query here, including complex ones.

SELECT * FROM myapp_person WHERE first_name = %s
params: SQL Query Parameters

Description: A tuple or dictionary containing the parameters that will be passed into the SQL query. This ensures safety from SQL injection attacks.

('John',)
translations: Field Renaming

Description: This optional parameter allows you to rename database fields to match your Django model fields. It's helpful when the database schema doesn't follow Django's convention.

{"first_name": "first_name_model"}
