Advanced Usage of Parameters with `params` in Django's `raw()` Queries

© Ruth Ewho

  • Parameterized Query with Safe Value: Avoid hardcoding values in queries to prevent SQL injection. Example: Person.objects.raw("SELECT * FROM myapp_person WHERE last_name = %s", [lname])
  • Multiple Parameters with Dictionary Style: Use dictionary-style placeholders to manage multiple parameters easily. Example: Person.objects.raw("SELECT * FROM myapp_person WHERE first_name = %(first)s AND last_name = %(last)s", {"first": "John", "last": "Doe"})
  • Dynamic Parameter Addition: You can easily add or remove parameters in a dictionary, especially useful for optional filters or user inputs. Example with multiple parameters: Person.objects.raw("SELECT * FROM myapp_person WHERE first_name = %(first)s AND last_name = %(last)s AND city = %(city)s", params)
