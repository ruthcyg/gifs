Using defer() for Specific Fields

The defer() method allows you to defer specific fields, loading others by default. This is ideal when you need most fields but want to exclude some.

# Defers 'account_history' and 'payment_info', loads others by default
customers = Customer.objects.defer("account_history", "payment_info")

Deferring fields minimizes the data retrieved initially, improving performance when large fields are unnecessary.

Using only() for Selective Loading

# Loads only 'id', 'first_name', 'last_name' immediately

This method ensures only essential fields are loaded, reducing query complexity and resource usage.
